# Student Note Taking App (NoteHub)

## Description
A web application for organizing university notes, featuring markdown support, attachment management, and study groups.

## Technologies
* **Frontend:** React.js
* **Backend:** Node.js / Express
* **Database:** MariaDB via Sequelize ORM
* **External API:** YouTube Data API
