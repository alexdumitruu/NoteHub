# Student Note Taking App (NoteHub)

## Description

NoteHub is a collaborative note-taking web application designed for university students.
Create and organize notes with Markdown support, attach files/images, and embed YouTube videos.
Share notes publicly in the Community or privately within Study Groups.
The app features a modern dark/light mode UI with real-time search and tag filtering.

## Technologies

- **Frontend:** React.js
- **Backend:** Node.js / Express
- **Database:** MariaDB via Sequelize ORM
- **External API:** YouTube Data API
